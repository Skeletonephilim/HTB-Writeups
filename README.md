# BLACK-BOX
This is a different repository from my Writeups repository even though it's also a Writeup repository, only I take the engagements here in a fully Black Box engagement type : no walkthroughs, no Guided Mode, no AI, just going blind. I'm authorized to do internet searches for patches/Vulnerabilities I find on the system, never to find the solution.
For example in my first ever Black Box engagement which was Optimum on HackTheBox, I found a certain patch on the systeminfo, and I had to search typing the name of the patch,
then website:microsoft.support.com to find the approximate date for the patch to find actual exploits that are unpatched on the system.

I use metasploit framework more openly during Black Box engagements than during engagements you can find in my Writeups because I have a lot more tools available in
the engagements that are found in my Writeups repository like VeXX, a specialized python scripter, and the engagements there are more Gray than Black because I can use
sources, writeups etc. when I'm stuck. Not here.
This is to test my limits as a pentesting operator and build a strong solo methodology, and also humble myself.
